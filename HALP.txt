Hello! This file details the custom bindings

Sections:
    NVIM
    SLIME
    FOLD
<Leader> is bound to both space and backslash

NVIM - basic custom keybinds
    <C-j> save the file and then enter the contents of the j register as a vim command, intended as a quick save and compile hotkey

SLIME - used for sending text to different tmux panes! Open up a target tmux and then send what you'd like to it!
    <Leader>-c   send the current cell
    <Leader>-v   send the current cell and move to the next
    <Leader>-a   send the current line
    <Leader>-l   send the configured repl_launch_command
    <Leader>-q   send the configured repl_exit_command
    <Leader>-p   send the configured repl_utility_command
    <Leader>-s   send a command you type
    <Leader>-:   create a cell delimiter
    <Leader>-j   save the file and then send the contents of the j register. This is useful when you want to save and run your files with a single hotkey!
    <Leader>-J   shortcut for writing something to the j register
    Recall: you can use ctrl-b-q to figure out what number each pane has
    The tmux command can create a target which nvim has been configured to send to by default:
        tmux -L nvim_target

FOLD - used to quickly create foldmarkers!
    <Leader>-' create a fold marker which has a name
    <Leader>-; create a unnamed foldmarker
Both of these hotkeys are for VISUAL LINE mode meaning they create folds surrounding a set of lines!

This nvim setup uses vim-plug. If you are having problems with plugins you can use :PlugInstall to make sure they are installed
