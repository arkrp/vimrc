GENERAL_INFO - details that aren't related to specific plugins. #section-start

<Leader> is bound to both space and backslash

This nvim setup uses vim-plug. If you are having problems with plugins you can use :PlugInstall to make sure they are installed

The normal commands zo and zc are used to expand and contract folds. That is the main feature that this config was built around. To see more check out FOLD

#section-end
NVIM - basic custom keybinds #section-start
    <C-j> save the file and then enter the contents of the custom_command varaible as a vim command, intended as a quick save and compile hotkey
#section-end
SLIME - used for sending text to different tmux panes! Open up a target tmux and then send what you'd like to it! #section-start
    <Leader>-c   send the current cell
    <Leader>-v   send the current cell and move to the next
    <Leader>-a   send the current line
    <Leader>-l   send the configured repl_launch_command
    <Leader>-q   send the configured repl_exit_command
    <Leader>-p   send the configured repl_utility_command
    <Leader>-s   send a command you type
    <Leader>-:   create a cell delimiter
    <Leader>-j   save the file and then send the contents of the custom_command variable. This is useful when you want to save and run your files with a single hotkey!
    <Leader>-J   shortcut for writing something to the custom_command variable
    Recall: you can use ctrl-b-q to figure out what number each pane has
    The tmux command can create a target which nvim has been configured to send to by default:
        tmux -L nvim_target
#section-end
FOLD - used to quickly create foldmarkers! #section-start
    <Leader>-' create a fold marker which has a name
    <Leader>-; create a unnamed foldmarker
Both of these hotkeys are for VISUAL LINE mode meaning they create folds surrounding a set of lines!
#section-end
NETRW - used to browse files #section-start
   ! (when netrw is open) - run command in directory that netrw is viewing!
   n (when netrw is open) - create new file (buffer) in directory
#section-end
FIND_AND_REPLACE - for when I forget again. #section-start
    Replace through the whole file:
        :%s/pattern_to_replace/replacement_text/g
    Replace in the last VISUAL selection:
        :%s/\%Vpattern_to_replace/replacement_text/g
#section-end
